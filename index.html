How I Work On Web Development

I begin my process with A LOT of procrastination. No, but seriously. I will get a great idea for an assignment or project and then put it off until the last second. It's a problem, but it's my problem. On the off chance that I make it past this stage, I like to do a few things. Mostly, I work best when I have a list of what I need to do. I'm nothing if not organized.

  • Normally, I start out by putting on some sort of playlist. I like to listen to instrumental stuff mostly while I'm working; it keeps me focused. Movie/video game soundtracks are what I lean towards on most days.
  • I always make a list of what my content is going to include in a journal.
  • Then, I sketch a couple different layouts for what the site will actually look like.
  • From here, I create the color scheme that I'm going to use, and I figure out which elements are going to be what.
  • Beginning the actual coding process, I start with a simple skeleton of my html with filler text. I get everything organized where I want it to be.
  • To finish it off, I work with css (and javascript if I have to, which I hope I rarely have to, because I hate it) to polish everything and make sure that it has the aesthetic that I want it to have.
